# Logo Móveis Planejados São José

Pasta com versões do logo para uso em sites parceiros (Escola Habilidade, etc.).

## Arquivos

| Arquivo | Quando usar |
|---|---|
| `logo-claro.svg` / `logo-claro-*.png` | **Fundo escuro** (texto branco) |
| `logo-escuro.svg` / `logo-escuro-*.png` | **Fundo claro** (texto preto + São José em verde) |
| `logo-cor.svg` / `logo-cor-*.png` | **Fundo claro** — versão monocromática verde marca (#1f3a34) |

## Tamanhos PNG

- `*-600.png` (600×200 px) — uso padrão em listas de parceiros / rodapé
- `*-1200.png` (1200×400 px) — uso em alta resolução / retina / hero

## Recomendação para site parceiro

HTML para usar na seção "Parceiros":

```html
<a href="https://www.movelplanejadosaojose.com.br" target="_blank" rel="noopener noreferrer">
  <img src="logo-cor-600.png" alt="Móveis Planejados São José — marcenaria sob medida em São José SC" width="300">
</a>
```

Use `logo-claro-600.png` se a seção de parceiros tiver fundo escuro.

## Especificações da marca

- **Cor primária (São José):** `#1f3a34` (verde escuro)
- **Tipografia:** Montserrat (Light 300 + Bold 700)
- **Proporção:** 3:1 (largura:altura)
- **Fundo:** transparente (PNG e SVG)
